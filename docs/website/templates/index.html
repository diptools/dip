Write cross-platform application with React-like declarative UI and scalable ECS architecture all in Rust.


dip is aiming to provide essential tools to build Web3 applications runs everywhere. (desktop, CLI, game etc.)


All plugins are based on Bevy ECS design pattern.

dip plugins are not only compatible with dekstop applications but also games like metaverse.

Features (Plugins)


  • dip CLI : Command line tools for dip developers.
  • DesktopPlugin : Tauri and Dioxus based plugin for Desktop apps. Imagine Electron + React but in Rust.
  • CliPlugin, SubCommandPlugin : clap based plugin for CLI apps.
  • ConfigPlugin
  • UiStatePlugin, UiActionPlugin : Redux inspired global state management plugin.
  • AsyncActionPlugin
  • BundlePlugin : Dotfiles made easy. One command to setup your laptop.
  • DevicePlugin : I'm experimenting with Ledger devices to manage keys.
